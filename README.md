## Play

***Move snake*** ***=>*** ``Up``, ``Down``, ``Left``, ``Right Arrow``, ``W``, ``A``, ``S``, ``D``

***Toggle Boost Mode*** ***=>*** ``Space``

***Toggle Sound*** ***=>*** ``M``

***Pause Game*** => ``P``

***To Quit*** => ``Q``
